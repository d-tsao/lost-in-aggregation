## About this repository 

This repository contains the R code for the simulations and figures in *Lost In Aggregation: The Causal Interpretation of the IV Estimand* (Tsao et al., 2026: https://arxiv.org/abs/2601.12120). [![DOI](https://zenodo.org/badge/1144370398.svg)](https://doi.org/10.5281/zenodo.18414840)

- `aggr_simulations.R`: provides all required code. 
- `fig`: contains all produced figures.
